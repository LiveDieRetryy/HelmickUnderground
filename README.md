# Helmick Underground LLC Website

Professional website for Helmick Underground LLC - Underground Utility Services

## About

This website showcases the services and contact information for Helmick Underground LLC, a professional underground utility company based in Mount Vernon, Iowa, serving customers since 1988.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern Layout**: Clean, professional design with smooth animations
- **Service Showcase**: Detailed information about all services offered
- **Contact Form**: Easy-to-use contact form for customer inquiries
- **Mobile Navigation**: Hamburger menu for mobile devices
- **SEO Optimized**: Proper meta tags and semantic HTML

## Setup Instructions

1. **Add Your Logo**: Place your logo image as `logo.png` in the same folder as the website files

2. **Files Included**:
   - `index.html` - Main website file
   - `styles.css` - Styling and design
   - `script.js` - Interactive features
   - `README.md` - This file

3. **To View Locally**:
   - Simply open `index.html` in any web browser

## Publishing Options

### Option 1: GitHub Pages (Free)
1. Create a GitHub account at https://github.com
2. Create a new repository
3. Upload all website files
4. Go to Settings > Pages
5. Select main branch and save
6. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify (Free)
1. Go to https://www.netlify.com
2. Sign up for free account
3. Drag and drop your website folder
4. Get instant live website with custom domain option

### Option 3: Traditional Web Hosting
1. Purchase domain name (e.g., helmickundergound.com)
2. Purchase web hosting (GoDaddy, Bluehost, HostGator, etc.)
3. Upload files via FTP or hosting control panel

## Customization

### Update Contact Email
In `script.js`, line 47, replace `info@helmickundergound.com` with your actual email address.

### Update Colors
In `styles.css`, modify the CSS variables at the top:
- `--primary-color`: Main orange color
- `--secondary-color`: Dark blue color
- `--dark-color`: Text color
